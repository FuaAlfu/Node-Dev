## Install typescript 
```
npm i --save-dev @types/mongoose
```

## to use typescript with node express
```
npm i --save-dev @types/cors @types/express @types/debug source-map-support tslint typescript
```

## install typescript
```
npm i -g typescript
```

## create typescript config file using ts compiler
```
tsc --init
```
