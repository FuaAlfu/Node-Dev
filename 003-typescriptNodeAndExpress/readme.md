## typescript - node with express

---
---

## install typescript
```
npm i -g typescript
```

## create typescript config file using ts compiler
```
tsc --init
```

## creating route for src folder
```
tsc
```

## Install 

---

## install json.pkg
```
npm init -y
```

## install express & nodemon
```
npm i express nodemon
```

## Install Dev - types
```
npm i -D typescript ts-node @types/node @types/express
```
